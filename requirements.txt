# ============================================================================
# IMPORTANT: Installation Instructions for Lambda Labs / Ubuntu
# ============================================================================
# Ubuntu systems often have system-installed PyTorch packages in
# /usr/lib/python3/dist-packages/ that can conflict with pip installations.
# 
# SOLUTION: Use a virtual environment to avoid conflicts.
#
# Quick Setup:
#   1. Create virtual environment:
#      python3 -m venv venv
#      source venv/bin/activate
#
#   2. Upgrade pip:
#      pip install --upgrade pip
#
#   3. Install PyTorch packages FIRST (must match CUDA version):
#      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
#      (For CUDA 12.1: --index-url https://download.pytorch.org/whl/cu121)
#
#   4. Install remaining requirements:
#      pip install -r requirements.txt
#
# Or use the setup script:
#   bash scripts/setup_lambda.sh
# ============================================================================
